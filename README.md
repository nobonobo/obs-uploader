# obs-uploader

An automated uploader that posts OBS recordings to YouTube upon file completion.

```
go install github.com/wailsapp/wails/v3/cmd/wails3@latest
```

```
wails3 task package
wails3 task build
./bin/obs-uploader
```


## prerequired tools

- ffmpeg
- ffprobe
- Inkscape
- youtubeuploader
- thumb-tool
